vis.graph_view — Graph Visualization¶

The vis.graph_view module supports visualization of candidate graph objects, nodes, and edges. This is basic visualization functionality that is monkey patched in to the graph objects. This is not intended to be low level diagnostic visualization

New in version 0.1.0.
