compute_triangular_cycles()[source]¶
Find all cycles of length 3. This is similar
to cycle_basis (networkX), but returns all cycles. As opposed to all basis cycles.
Returns:cycles – A list of cycles in the form [(a,b,c), (c,d,e)], where letters indicate node identifiers
Return type:list

Examples

>>> g = CandidateGraph()
>>> g.add_edges_from([(0,1), (0,2), (1,2), (0,3), (1,3), (2,3)])
>>> g.compute_triangular_cycles()
[(0, 1, 2), (0, 1, 3), (0, 2, 3), (1, 2, 3)]
